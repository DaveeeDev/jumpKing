# Jump King Mobile Clone and PC Port (Godot 2.5D)

This is a mobile clone of the game *Jump King*, which unfortunately is no longer available on the Google Play Store. It is my first real 2.5D game developed in a 3D-capable engine, and it's already mostly finished!

---

## Gameplay & Features

The core mechanic is simple but addictive: you tap either **left/A** or **right/D** to follow a randomly generated zigzag staircase and climb as high as you can, as fast as you can.

### What's working:
* **Currency System:** Earn rewards as you climb.
* **Skin System:** Spend your coins on cosmetic upgrades.
* **Quest System:** In-game challenges to complete.
* **Fixed Mode:** The base game mode is fully playable.

### What's still missing / WIP:
* **Endless Mode:** To see how high players can truly go.
* **Hard Endless Mode:** For those who want an extra punishing challenge.
* **UI/UX Overhaul:** The menus, buttons, sprites, and progress bars are currently just barebones placeholders for testing and need a complete visual redesign.
* **Skin Expansion:** The current skins are quickly put together and need high-quality replacements.
* **More Quests:** While the quests are already loaded from a config file for easier implementation, the config file needs a way bigger array of quests. The tracking of various statistic has to be hardcoded however.

## Tech Stack & Learning
* **Engine:** Godot
* **Style:** 2.5D (Isometric/Orthographic feel using Godot's engine capabilities)
* **Status:** Mostly finished backend, UI and extra modes are Work in Progress (WIP).
